Privacy Policy — Sumo Squared Arena

Last updated: June 30, 2026

Sumo Squared Arena is operated by a single individual as a personal automated video-publishing tool. It is not a multi-user service — it does not have visitors, accounts, or sign-ups of its own.

Data accessed

Via each platform's official API (e.g. TikTok's Content Posting API), this tool accesses only:

  • An OAuth access token and refresh token, issued directly by the platform to the operator's own account, used solely to authorize video uploads.
  • The video file the tool itself generates, uploaded as a post to the operator's own account.

Data storage

Access and refresh tokens are stored locally on the operator's own machine, in a file that is never committed to version control or shared with any third party. Video files are stored only temporarily during the upload process and then deleted.

Data sharing

No data is sold, shared, or transmitted to any third party. Tokens are used exclusively to call the issuing platform's own API.

Data retention

Tokens persist locally until manually revoked by the operator (via the platform's own account settings) or replaced. No other user data is retained, because this tool has no end users of its own.
